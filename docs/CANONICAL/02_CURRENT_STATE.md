# Current State

Piku is in active development. Core infrastructure is built:
- Ollama-based local LLM inference (qwen3:4b default, qwen3:14b for heavy reasoning)
- nomic-embed-text for embeddings
- IndexedDB-based knowledge graph with 9 node types and 10 edge relationships
- Active-app observer for ambient context gathering
- Immersive chat UI
- Persistence layer (IndexedDB v6)
